jesgoo_interface.resizeContainer(height, isFixed)

@description 设定容器的高度
@param {number} height 高度，单位px
@param {boolean} isFixed 固定高度，即不随同设备大小而自动变化的固定高度

jesgoo_interface.autoResizeImage(isFixed)

@description 根据ImageUrl提供的图片自动设定容器高度，再图片横向撑满容器时，容器置为图片最适高度，防止伸缩变形
@param {boolean} isFixed 固定高度，即不随同设备大小而自动变化的固定高度
